## Final project on algorithms

## Аналіз ймовірностей сум при киданні двох кубиків (Метод Монте-Карло)

Метою є обчислення ймовірностей випадіння різних сум чисел при киданні двох
гральних кубиків за допомогою методу Монте-Карло (імітаційне моделювання).
Результати симуляції порівнюються з теоретично розрахованими значеннями.

### Результати симуляції

Було проведено **1,000,000** імітацій кидків. Нижче наведено порівняльну таблицю
отриманих емпіричних даних та теоретичних розрахунків.

|  Сума  | Імовірність (Монте-Карло) | Теоретична Імовірність | Відхилення |
| :----: | :-----------------------: | :--------------------: | :--------: |
| **2**  |           2.78%           |      2.78% (1/36)      |   ~0.00%   |
| **3**  |           5.56%           |      5.56% (2/36)      |   ~0.00%   |
| **4**  |           8.34%           |      8.33% (3/36)      |   +0.01%   |
| **5**  |          11.10%           |     11.11% (4/36)      |   -0.01%   |
| **6**  |          13.90%           |     13.89% (5/36)      |   +0.01%   |
| **7**  |          16.66%           |     16.67% (6/36)      |   -0.01%   |
| **8**  |          13.88%           |     13.89% (5/36)      |   -0.01%   |
| **9**  |          11.12%           |     11.11% (4/36)      |   +0.01%   |
| **10** |           8.33%           |      8.33% (3/36)      |   ~0.00%   |
| **11** |           5.55%           |      5.56% (2/36)      |   -0.01%   |
| **12** |           2.78%           |      2.78% (1/36)      |   ~0.00%   |

### Висновки

1. **Точність методу**: Результати, отримані за допомогою методу Монте-Карло, є
   дуже близькими до аналітичних розрахунків. Відхилення складають соті частки
   відсотка, що є допустимою похибкою для статистичного моделювання.
2. **Закон великих чисел**: Експеримент підтверджує, що при великій кількості
   повторень (1 млн) частота події наближається до її теоретичної ймовірності.
3. **Розподіл**: Найбільш вірогідною сумою є **7** (16.67%), а найменш
   вірогідними - **2** та **12** (2.78%), що повністю відповідає формі
   нормального розподілу для дискретних величин.

Розрахунки є правильними та статистично значущими.
